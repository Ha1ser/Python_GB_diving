**–ö–æ–º–∞–Ω–¥–∞ yield** - –ö–æ–º–∞–Ω–¥–∞ yield —Ä–∞–±–æ—Ç–∞–µ—Ç –∞–Ω–∞–ª–æ–≥–∏—á–Ω–æ return. –ù–æ –≤–º–µ—Å—Ç–æ –∑–∞–≤–µ—Ä—à–µ–Ω–∏—è —Ñ—É–Ω–∫—Ü–∏–∏ –∑–∞–ø–æ–º–∏–Ω–∞–µ—Ç –µ—ë —Å–æ—Å—Ç–æ—è–Ω–∏–µ.
–ü–æ–≤—Ç–æ—Ä–Ω—ã–π –≤—ã–∑–æ–≤ –ø—Ä–æ–¥–æ–ª–∂–∞–µ—Ç –∫–æ–¥ –ø–æ—Å–ª–µ yield.
```Python
def factorial(n):
    number = 1
    for i in range(1, n + 1):
        number *= i
        yield number

for i, num in enumerate(factorial(5), start=1):
    print(f"{i}! = {num}")
# 1! = 1
# 2! = 2
# 3! = 6
# 4! = 24
```

```Python
def factorial(n):
    number = 1
    for i in range(1, n + 1):
        number *= i
        yield number

my_iter = iter(factorial(4))

print(next(my_iter)) # 6
print(next(my_iter)) # 24
print(next(my_iter)) # StopIteration
```
